# YouStory

> 把你人生故事里的某个片段，具象化成看得见的画面。

YouStory 是一个用于生成人生片段影像的创作工具。你可以描述脑海中的某个场景、某段回忆或某个故事片段，YouStory 会根据你的描述生成对应的图片，并进一步把这些图片自动组合成一段属于你的人生片段视频。

它关注的不是把完整人生写成宏大的传记，而是捕捉那些具体、真实、值得被留下来的瞬间：一次告别、一场旅行、一个雨夜、一次谈话，或某个突然想起的普通下午。

## What You Can Do

### 片段描述生成图片

用自然语言描述你脑海中的画面，YouStory 会将描述转化为更具象的视觉结果。

你可以描述：

- 场景：地点、时间、天气、光线、氛围
- 人物：外貌、动作、表情、关系
- 情绪：怀念、快乐、遗憾、平静、紧张
- 画面风格：写实、电影感、漫画感、插画感

### 多张图片组成故事片段

一个人生片段通常不只是一张图。YouStory 可以根据你的描述生成多张连续图片，让它们共同表达一个短场景。

例如：

- 第一次离开家去外地上学
- 和朋友在深夜便利店聊天
- 旅行途中突然下雨
- 某个改变关系的沉默瞬间

### 图片自动生成片段视频

生成的图片可以继续被组织成一段短视频。视频可以包含画面顺序、镜头节奏、转场、字幕或旁白，用更接近“人生片段”的方式保存你的故事。

### 用照片辅助还原人物

当你希望画面中的人物更贴近真实记忆时，可以提供人物照片作为参考，让生成结果更接近你想表达的人和场景。

## Example Use Cases

- 把日记里的一段回忆变成图片
- 把旅行中的一个瞬间生成短视频
- 为朋友或家人制作一段纪念影像
- 将脑海中的画面快速转成视觉草图
- 为个人故事、播客、文章或社交内容生成配图

## Getting Started

阅读本文档即可了解 YouStory 的产品定位、核心功能和使用方式。

推荐的使用流程如下：

1. 准备一个具体片段，不需要完整故事。
2. 用自然语言写下画面描述。
3. 补充人物、场景、情绪和风格要求。
4. 生成一组图片。
5. 将图片自动组合成片段视频。

示例输入：

```text
一个夏天傍晚，我和高中朋友坐在操场边的台阶上。
天快黑了，远处有人在打篮球，风很轻。
我们没有说太多话，但都知道以后很难再像这样见面。
画面希望有一点怀旧、安静、电影感。
```

## Technology Stack

YouStory 采用双运行模式：

- `Cloud Demo`：部署到 Vercel，支持文字生成、图片场景生成、图片保存与展示。
- `Local Full`：用户本地运行，支持文字、图片、视频、音频的完整生成与管理。

核心技术栈：

- 前端：Next.js + React + TypeScript + Tailwind CSS + shadcn/ui
- 后端：Node.js 20+ + TypeScript + Hono
- Cloud：Neon Postgres + Vercel Blob images only
- Local：SQLite + local filesystem + FFmpeg
- AI 服务适配：文字、图片、视频、音频

完整技术方案见 [docs/tech-stack.md](docs/tech-stack.md)。

## Series Workflow

YouStory 的下一阶段会把一个项目视为一部完整剧集：项目下包含多集，每一集都有剧本创作、分镜设计、视频生成、配音及音效、后期剪辑五个环节。

完整功能计划见 [series-project-workflow.md](series-project-workflow.md)。

## Deployment

YouStory 的部署目标分为两类：

- Vercel 部署：用于公开展示和轻量体验，不处理视频生成与视频存储。
- 本地运行：用于完整创作流程，包含本地文件存储和视频生成。
