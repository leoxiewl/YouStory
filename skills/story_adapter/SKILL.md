# Story Adapter — 个人故事转化为短剧剧本

## 核心任务

将用户的个人生活记录（日记、随感、回忆、旅行笔记等）转化为适合拍摄的短剧剧本格式。

## 转化原则

### 1. 保留情感真实性
- 不要过度戏剧化，保持原始情感的真实质感
- 维持记录者的第一人称视角和个人声音
- 真实的生活细节比虚构的戏剧冲突更有力量

### 2. 识别可视化时刻
从文字中找到可以被具象化为镜头的关键场景：
- 具体的地点和环境描述
- 人物互动和对话
- 情绪转折点（发现、领悟、告别、重逢）
- 感官细节（味道、声音、光线、触感）

### 3. 故事类型策略

**日记（diary）**：保留现在时态的观察感，以旁白为主，对话为辅
**随感（reflection）**：内心独白 + 闪回场景，配合慢镜头描述
**回忆（memory）**：过去时态，使用"记忆质感"的画面描述（稍模糊/温暖色调）
**旅行（travel）**：地理移动 + 遭遇，配合空镜和氛围镜头
**家人（family）**：日常生活场景，重视细节和情感积累
**成长（growth）**：时间跨度叙事，可使用"当时的我"vs"现在的我"对比结构

### 4. 剧本格式要求

```
【第X集：标题】

[场景：地点描述，时间，氛围]
[旁白（可选）]：...
[人物名]（动作描述）：对白内容
[旁白（可选）]：...

[下一场景]
...
```

### 5. 输出长度控制
- 每集剧本控制在 500-1500 字
- 对应 5-15 个可拍摄场景
- 每个场景对应 1-2 个分镜镜头
